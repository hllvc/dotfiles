# .config

Application configuration files managed via GNU Stow.

[Back to root](../)

## Directories

| Directory | Description |
|-----------|-------------|
| `1Password/` | SSH agent configuration for 1Password integration |
| `ghostty/` | Ghostty terminal configuration (modular: font, theme, UI, quality-of-life) |
| `karabiner/` | Keyboard remapping rules (Caps Lock to Escape, vim-style navigation) |
| `launch-agents/` | macOS LaunchAgents — tmux sockets (`com.hllvc.{work,personal}.tmux`) and the `com.hllvc.memory-pressure` monitor. Loaded via `./dotctl agents load`. |
| [`nvim/`](nvim/) | Neovim configuration with lazy.nvim, LSP, snippets, and custom keymaps |
